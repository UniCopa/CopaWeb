Privilegien

Es gibt folgende Rechtegruppen: "Eigner","Stellvertreter" und "Rechteinhaber". Ein Nutzer kann bezüglich einer Veranstaltung zu genau einer dieser Rechtegruppen gehören. Abhängig von der Rechtegruppe kann man verschiedene Aktionen ausführen.:

Eigner
Ein Eigner ist vom CoPA-System oder einem Administrator bestimmt worden.
Als Eigner kann man Stellvertreter oder Rechteinhaber auf der CoPA-Webseite ernennen oder entfernen. Außerdem können Eigner Termine von ihren Veranstaltung ändern und auch die anderen Eigner, Stellvertreter oder Rechteinhaber einsehen.
Für einen Eigener ist es nicht möglich seine Rechte abzugeben.

Stellvertreter
Ein Stellvertreter ist bezüglich einer Veranstaltung von einem Eigener ernannt worden.
Ein Stellvertreter kann auf der CoPA-Webseite Rechteinhaber bestimmen oder entfernen und auch seine eigenen Rechte abgeben. Ein Stellvertreter kann ebenfalls alle Eigner, Stellvertreter, auch Rechteinhaber von seiner Veranstaltung einsehen und einen Termin der Veranstaltung ändern.

Rechteinhaber
Ein Rechteinhaber wird von einem Eigner oder Stellvertreter für die jeweilige Veranstaltung ernannt.
Ein Rechtinhaber kann auf der CoPA-Website seine Rechte aufgeben.Außerdem kann er einen Termin einer Veranstaltung verändern. Allerdings ist es ihm nicht möglich Eigner, Stellvertreter oder Rechteinhaber zu ernnenn.

Funktionalität

Folgende Funktionen sind mit der CoPA-Website nutzbar:

Anmelden
Nutzen Sie ihre Uni-Anmelde-Daten um sich zu authentifizieren.

Abmelden
Den Weg zur Abmelde-Seite finden Sie indem sie den Zahnrad-Knopf nutzen und anschließend auf "Logout" klicken, wenn Sie bereits angemeldet sind.

Suchen
Den Weg zur Suchen-Seite finden Sie in der Funktionsleiste am linken Bildschirmrand. Dort können sie sich, durch Auswahl der entsprechenden Kategorie, das Event ihrer wahl aussuchen.

Abonnieren
Um eine Veranstaltung zu abonnieren, nutzen Sie zunächst die Suche um diese zu finden.Anschließend können Sie auf den "Abonnieren"-Knopf klicken. Nachdem Sie die Veranstaltung abonniert haben, werden sie über Änderungen an Terminen dieser Veranstaltung informiert. Die Art der Information hängt von den von Ihnen gewählten Einstellungen ab.

Abbestellen
Um eine Veranstaltung abzubestellen, suchen Sie diese zunächst in Ihrer Liste der abonnierten Veranstaltungen (siehe "Abonnierte Veranstaltungen ansehen") und klicken auf den "Abbestellen"-Knopf. Oder sie verwenden wieder die Suchfunktion und navigieren mit dieser zum gewünschten Event um dieses abzubestellen. Anschließend werden Sie nicht länger über änderungen informiert.

Abonnierte Veranstaltungen ansehen
Mit einem Klick auf "Meine Abos" in der Funktiosleiste, gelangen Sie auf die Seite Ihrer abonnierten Veranstaltungen.

Informationstext zu einer Veranstaltung ansehen
Informationen zu einer Veranstaltung finden sie indem sie direkt auf eine Veranstaltung klicken.

Veranstaltungen mit besonderen Rechten ansehen
Mit einem Klick auf Rechteverwaltung, in der Funktionsleiste, können sie alle Veranstaltungen zu denen sie besondere Rechte haben ansehen.

Eigner,Stellvertreter und Rechteinhaber einer Veranstaltung ansehen
Diese können Sie sich nur ansehen, wenn Sie besondere Rechte bezüglich dieser Veranstaltung haben. Klicken Sie in der Liste Ihrer Veranstaltungen mit besonderen Rechten auf den vorrangestellten Bearbeitenbutton einer Veranstaltung um sich die jeweiligen Nutzer anzeigen zu lassen.

Einen Termin einer Veranstaltung ändern
Zunächst müßen Sie den Termin, den sie ändern wollen, finden. Nutzen sie dazu den Link "Kurse ändern" und wälen sie sich eine entsprechende Veranstaltung, über den Bearbeitenbutton aus. Danach muss noch der entsprechende Termin ausgwählt werden und sie können die Änderung eintragen

Farbe einer Veranstaltung ändern
Klicken Sie in der Liste der abonnierten Veranstaltungen auf den "Farbe"-Knopf und wählen Sie die neue Farbe aus.

E-Mail-/CoPA-Webseiten-Sprache ändern
Klicken Sie auf den Zahnrad-Knopf um die Sprache und die E-Mail einzuststellen. Für weitere Informationen zu den Auswahlmöglichkeiten lesen Sie "mögliche Einstellungen"

Benachrichtigungsart ändern
Klicken Sie auf den Zahnrad-Knopf um die Benachichtungsart einzustellen. Für weitere Informationen zu den Auswahlmöglichkeiten lesen Sie "mögliche Einstellungen".


Folgende Funktionen sind nur von der CoPA-Webseite zu erreichen:
Rechte aufgeben
Sie können ihre Vorteile als Rechteinhaber oder Stellvertreter auf der Webseite aufgeben.

Ernennen oder Entlaßen von Stellvertretern und Rechteinhabern
Abhängig von der Rechtegruppe, der Sie angehören(siehe "über Rechte"), können sie anderen Rechte erteilen oder entziehen.

Einstellungen

Sprache
E-mail und CoPA-Webseite:
In den Einstellungen können sie zwischen Englisch und Deutsch wählen. Falls Sie E-Mail als Benachrichtigungsart ausgewählt haben, erhalten Sie diese in der hier von Ihnen ausgewählten Sprache.
Auch die CoPA-Webseite wird diese Sprache verwenden, sobald Sie sich angemeldet haben.

CoPA-App
Die CoPA-App kann in zwei Sprachen genutzt werden: Deutsch oder Englisch. Sie paßt sich automatisch an ihre Systemsprache an. Wenn die Systemsprache weder deutsch noch englisch ist, wird die App englisch verwenden. Um die Sprache zu verändern, sehen Sie in Ihre System-Einstellungen.

Benachrichtigungsart
E-Mail
Wenn Sie dies aktivieren, wird Ihnen automatisch eine E-Mail zugeschickt, wenn sich eine Ihrer abonnierten Veranstaltungen verändert. Dazu wird Ihre Uni-E-Mail-Adreße verwendet.

Automatisch pushen
Wenn Sie diese Einstellung wählen, erhalten Sie, sobald sich etwas ändert, eine Nachricht auf Ihr Gerät und die änderung wird sofort auf ihrem Gerät angezeigt. Natürlich können Sie die Nachricht und die änderung nur erhalten, wenn Sie mit dem Internet verbunden sind.

Manuell pushen
Wenn Sie diese Einstellung wählen, erhalten Sie, sobald sich etwas ändert, eine Nachricht auf Ihr Gerät, aber die änderung wird nicht sofort auf ihrem Gerät angezeigt. Um die änderung zu holen, müßen Sie den "Aktualisieren"-Knopf auf der Startseite nutzen. Natürlich können Sie die Nachricht nur erhalten, wenn Sie mit dem Internet verbunden sind.
